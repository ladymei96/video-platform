YouTube DATA API 格式：https://www.googleapis.com/youtube/v3/

1.取得 單支影片 相關資訊 API
https://www.googleapis.com/youtube/v3/videos?id=${影片id}&key=${金鑰}&part=snippet

-參數 part 說明：想在 JSON 內看到的內容

2.取得某頻道內影片清單之個別影片資訊

2-1.取得 Channel 內上傳影片清單的 playlists ID
    (透過回傳 JSON 可找到 uploads: playlists ID)

透過 channels list API 取得 Channel 相關資訊
https://www.googleapis.com/youtube/v3/channels?part=contentDetails&id=${頻道id}&key=${金鑰}

-頻道id：透過YT網址  https://www.youtube.com/channel/${頻道id} 取得

2-2. 利用上步驟取得之 playlists ID
    (透過回傳 JSON 可找到 個別影片之資訊)

透過 playlistItems list API 可取得它的影片清單
https://www.googleapis.com/youtube/v3/playlistItems?part=snippet,contentDetails,status&playlistId=${playlistsID}&key=${金鑰}&maxResults=${數量}

-參數 maxResults 說明：影片數量，最多50
-回傳 JSON 內 thumbnails 欄位有各種尺寸的影片縮圖網址
-回傳 JSON 內 nextPageToken 欄位 可以取得下一頁影片

2-3. 利用上步驟取得之 nextPageToken 帶入 playlistItems list API 內參數 pageToken
https://www.googleapis.com/youtube/v3/playlistItems?part=snippet,contentDetails,status&playlistId=${playlistsID}&key=${金鑰}&maxResults=50&pageToken=${剛剛取得之nextPageToken}

3. 利用影片 videoId 可以取得影片留言清單
https://www.googleapis.com/youtube/v3/commentThreads?part=snippet,replies&videoId=${videoId}&key=${金鑰}

-回傳 JSON 內包含內容、時間、留言者的名字跟大頭貼

4. Search影片
https://www.googleapis.com/youtube/v3/search?part=snippet&q=${搜尋的字串}&key=${金鑰}&type=video&maxResults=${數量}
